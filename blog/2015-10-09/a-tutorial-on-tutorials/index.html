Preface

There are varying levels of tutorials in iOS apps, from something as simple as an annotated screenshot on the App Store listing to a complicated, very closely integrated set of puppet actions that walk users through your app, highlighting one bit of functionality at a time.

As a user I don't really like the first one, and as a developer I really don't want to implement the second one. But most of the time, a series of images or screenshots with some accompanying text is good. Usually you swipe various pages (or skip them if you know how to use the app) and then proceed to the first screen in the app. This seems to be an acceptable not-lazy-but-also-easy solution to adding tutorials to your app (especially feature-specific tutorials, since users tire of swiping through 50 screens to get to their content).

What I'm going to show you is One Weird Trick to make these tutorials that won't require much effort from you.

PXTutorialManager

So let's role play. You be an app developer that wants to put a tutorial in a mostly complete app. I'll be your magical helper elf and make this an easy task to complete.

Okay dev, there are some things I need from you:

  • Some screenshots of your app (or any other image-based content you want to highlight as a part of your tutorial)
  • Text descriptions for each of these images
  • The ability to write a simple JSON file

See, that wasn't so hard. Oh wait, I'm getting ahead of myself. Now, make me a JSON file called APPTutorial.json that looks like this:

{
    "storageVersion" : 1.0,
    "pageArray" : [
        {
            "pageTitle" : "Welcome to APP",
            "pageText" : "This is the best feature of APP, and presumably the reason you downloaded it.",
            "pageImageName" : "bestFeatureOfAPPScreenshot"
        },
        {
            "pageTitle" : "Get Started with APP",
            "pageText" : "This is an example of how to use APP.  You will love it!",
            "pageImageName" : "usingAPPExampleScreenshot"
        },
        {
            "pageTitle" : "APP is Great",
            "pageText" : "Now that you've seen how to use all the FEATURES in APP, go forth and enjoy APP!",
            "pageImageName" : "greatAPP"
        }
    ]
}

Alright. Now you are done.

What? You want to know what this has to do with making a tutorial?

Well I'll tell you, Dmitri, you've given me all the information I need to make your tutorial. Yes. That was a nice process. I agree. Yes it was simple, Dmitri. I'm glad you thought so too. Alright. Yes, it's good that we both agree that this was easy.

There's only one more step you need to complete. Put this in your code where you want to display the tutorial:

[PXTutorialManager presentTutorial:@"APPtutorial" force:TRUE];
