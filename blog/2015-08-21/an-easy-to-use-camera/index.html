It's here. An easy library for getting user photos that does everything in one method call:

[[PXCamera camera] getImageInViewController:self interface:PXCameraInterfaceCamera completion:^(UIImage * image, PXCameraImageSource source) {
    // boom!
}];

A bit of history

Once upon a time several years ago, Pete Shirley, a friend of the founders of our company, approached us to help him try out some ideas he had in app form. He wanted to build an application for filtering selfies to look more halloween-like: he wrote some pretty involved filtering for making people look like vampires, ghouls, etc.. This being a sort of prototype/side-project-for-a-friend, development was more fast and loose with less regard for modularity than we usually like.

Meanwhile, I was working on another app for Pete, that centered around photo messaging. While writing this, I had to solve some of the same camera-related problems that we had already solved (albeit not in a transportable way) on the previous app. So I wrote a camera library in the way I felt it should work. All the complicated bits and state keeping were hidden behind a simple interface of the form give-me-an-image-from-this-source.

Fast forward a year, this is now how all of the apps in the Pete Shirley family of apps get their photos from this camera library. The original code in the first filtering projects we did has since been upgraded to use this library and maintainability has skyrocketed.

PXCamera

There are a couple of things that this camera provides that are nice beyond what the UIImagePickerViewController gives you:

  • An optional grid overlay (like the full camera app)
  • A button to pick from the photo library in case the user opens the camera and then decides they wanted a picture they already took
  • A shutter delay timer (long press the shutter button) for taking timed photos
  • A really simple Objective-C interface
 

The default Apple camera is pretty Spartan and it's no fun to set up. PXCamera tries to be the opposite on both counts. Now you can use this pod and have your user taking or picking photos in no time at all. Enjoy!
